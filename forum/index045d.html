Supported AC:
• Valve Anticheat 3 - Secured ✔
• Live Anticheat - Secured ✔

Supported Games:
• Call of Duty: Ghosts Multiplayer (Steam v3.15.2)
• Call of Duty: Ghosts Multiplayer (PlusIW6 v3.15.2)

Aimbot: (Works with Gryphon, Helo Pilot, Loki and Odin)
• Aimtype - [Off / Manual / Automatic]
• Autozoom - [Off / On]
• Autofire - [Off / On] (100% compatible with akimbo weapons)
• Autowall - [Off / On]
• Silent-Aim - [Off / On] (non-pulling, server-sided, invisible to spectators and on killcam, 100% compatible with akimbo weapons)
• Target Agents - [Off / On] (lock on to guard dogs, squadmates, etc.)
• Target Missiles - [Off / On] (lock on to I.E.D.s and C4)
• Bonescan - [Off / On Timer / Immediate]
• Anti-Aim - [Off / Spinbot / Jitterbot / Randomized / Reversed] (makes it harder to be hit by an opposing aimbot)

Wallhack:
• Player Boxes - [Off / Border / Shadow]
• Player Corners - [Off / Colored / Black]
• Player Bones - [Off / Skeleton / Dots]
• Player Snaplines - [Off / On]
• Player Info - [Off / On] (distance, name)
• Player Weapons - [Off / On] (shader icons)
• Agents - [Off / On] (shader icons)
• Missiles - [Off / On] (shader icons)
• Items - [Off / On] (shader icons)

Miscellaneous:
• No Recoil - [Off / On]
• No Spread - [Off / On] (server-sided, invisible to spectators and on killcam,
100% compatible with akimbo weapons, works in any zoom state)​
• Killspam - [Off / On]
• Name Stealer - [Off / On]
• Crosshair - [Off / On]
• Compass - [Off / On]
• Radar - [Off / On] (displays a movable 2D radar window)
• Console - [Off / On] (displays a command input window)
• Player List - [Off / On] (displays a window that lists every player and some options/info for them)
• Host Menu - [Off / On] (displays a window with editable info for all players in the lobby, requires host)
• Riotshields - [Off / Ignore Player / Target Feet]

Tweaks:
• Aimbone - [Head / Neck / Upper Spine / Lower Spine / Left Shoulder / Right Shoulder / Left Hip / Right Hip /
Left Elbow / Right Elbow /Left Knee / Right Knee / Left Wrist / Right Wrist / Left Ankle / Right Ankle] (bonescan must be off)​
• Aim Strength - [5% to 100%] (power of the aimbot)
• Autoaim Time - [0 ms to 1000 ms] (transition time from current angles to destination angles)
• Autoaim Delay - [0 ms to 1000 ms] (time between target aquisition and autoaim)
• Autozoom Delay - [0 ms to 1000 ms] (time between target aquisition and autozoom)
• Autofire Delay - [0 ms to 1000 ms] (time between target aquisition and autofire)
• Visibility - [5 degrees to 360 degrees] (only target enemies within this fov)
• Field of View - [65 to 120]
• Frames per Second - [0 to 300] (0 means fully uncapped)
• Reset to Default - (sets tweaks tab settings to their default values)

Colors: (RGB)
• Enemy Visible
• Enemy Invisible
• Friendly Visible
• Friendly Invisible
• Entity Visible
• Entity Invisible
• Item Visible
• Item Invisible
• Crosshair
• Reset to Default - (sets colors tab settings to their default values)

Main Interface:
• Ping Display
• Hosting/Not Hosting Display
• Clickable & Moveable
• Set as Default - (saves settings to a default profile)
• Save Profile - (saves settings to a profile via save as file dialog)
• Load Profile - (loads settings from a profile via open file dialog)

Console: (Case Insensitive)
• rc_crash - (Crash everyone in the lobby except yourself)
• rc_name - <name> (Change your name)
• rc_compass - <on|off> (Enable/disable enemies on the minimap)
• rc_thirdperson - <on|off> (Enable/disable thirdperson view)
• rc_hostawall - <on|off> (Enable/disable host autowall (as host))
• rc_rapidfire - <on|off> (Enable/disable rapidfire weapon rate (as host))
• rc_chatspam - <on|off> <message> (Enable/disable custom chatspam message)
• rc_masskill - <on|off> <axis|allies|all> (Enable/disable player masskill (as host))
• rc_experience - <all|index> <max|experience> (Set your experience)
• rc_prestige - <max|number> (Set your prestige)
• rc_squadpoints - <max|squadpoints> (Set your squadpoints)
• rc_unlockall - (Unlock everything in the game)
• rc_resetstats - (Completely erase your save game)
• rc_hostdvar - <dvar> <value> (Set DVAR value for all clients (as host))

Player List:
• Targetlist - (Aimbot will only target selected players, at least one player must be checked)
• Crash - (crash only the player in question, requires host)
• IP Address - (IP address for the player in question, can be copy and pasted)

Host Menu: (Options Per Player)
• Player Select
• God Mode - [Off / On]
• Infinite Ammo - [Off / On]
• Explosive Bullets - [Off / On]
• Freeze Player - [Off / On]
• Name - (editable)
• Team - (editable)
• Health - (editable)
• Weapon - (select the weapon to give the specified player)
• Give Weapon
• Take Weapon
• Position - (editable)
• Teleport to Player - (teleport yourself to the specified player)
• Teleport Player From - (teleport the specified player to yourself)
• Kick this Player - (kick the specified player from the game)
• Reason - (reason for kick, optional)

Accelerators:
• Toggle Menu - [Insert Key]
• Force Host - [Delete Key] (force the game to start with you as host)
• Load Default Profile - [Home Key] (load settings from the default profile)
• Disable All - [End Key] (turns everything off instantly)
• Save Position - [Page Up Key] (save your current position, requires host)
• Load Position - [Page Down Key] (teleport back to your saved position, requires host)
